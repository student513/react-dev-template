# React Template

- typescript, babel, webpack, eslint가 세팅된 React 개발 템플릿입니다.

## Webpack Setting

- Typescript
- CSS/SCSS
- 절대경로 적용: `@/components/~~`
- 개발용 `webpack.config.dev.js`와 배포용 `webpack.config.prod.js` 설정 파일 분리

## Directory

- dist: 빌드 파일 생성 위치
- public: 템플릿 html 위치
- src: 소스 코드

## Eslint

- airbnb rule

## Command

- build
  `$ yarn build`
- start(port: 9000)
  `$ yarn start`
